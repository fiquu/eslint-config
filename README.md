# @fiquu/eslint-config
fiquu ESLint standard configuration.

## Usage
```
npm i -D @fiquu/eslint-config
```

### JavaScript
Extend `@fiquu` in your ESLint config.

#### Example

##### `.eslintrc`:
```json
{
  "extends": ["@fiquu"]
}
```

### TypeScript
Extend `@fiquu/eslint-config/typescript` in your ESLint config.

#### Example

##### `.eslintrc`:
```json
{
  "extends": ["@fiquu/eslint-config/typescript"]
}
```
